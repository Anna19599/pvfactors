base.BasePVArray¶

class base.BasePVArray(list_pvrows=[], ground=None, distance=None, height=None)[source]¶

Base class for PV arrays in pvfactors. Will provide basic capabilities.

__init__(list_pvrows=[], ground=None, distance=None, height=None)[source]¶

Initialize PV array.

Parameters:
  • list_pvrows (list of PVRow, optional) – List of PV rows in the PV array (Default = [])
  • ground (PVGround, optional) – Ground geometry for the PV array
  • distance (float, optional) – Unique distance between PV rows, if exists (Default = None)
  • height (float, optional) – Unique height of all PV rows, if exists (Default = None)
