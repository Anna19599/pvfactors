calculator.VFCalculator¶

class calculator.VFCalculator(mapper=None, vf_ts_methods=None)[source]¶

This calculator class will be used for the calculation of view factors for PV arrays

__init__(mapper=None, vf_ts_methods=None)[source]¶

Initialize the view factor mapper that will be used.

Parameters

  • mapper (VFMapperOrderedPVArray, optional) – View factor mapper, which will map elements of the PV array view matrix to methods calculating view factors (Default = VFMapperOrderedPVArray)

  • vf_ts_methods (VFTsMethods object) – Object with methods to calculate timeseries view factors for the fast mode (Default = None)

Methods
