calculator.VFCalculator¶

class calculator.VFCalculator(mapper=None)[source]¶

This calculator class will be used for the calculation of view factors for PV arrays

__init__(mapper=None)[source]¶

Initialize the view factor mapper that will be used.

Parameters:mapper (VF mapper object, optional) – View factor mapper, which will map elements of the PV array view matrix to methods calculating view factors (Default = VFMapperOrderedPVArray)

Methods

__init__([mapper]) Initialize the view factor mapper that will be used.
get_vf_matrix(geom_dict, view_matrix, …) Calculate the view factors based on the surfaces of the PV array object, and using a mapping of “view types” in the view matrix, to the view factor calculation methods.
