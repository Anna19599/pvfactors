pvfactors.pvrow.PVRowBase¶

class pvfactors.pvrow.PVRowBase[source]¶

PVRowBase exists for future developments of the model. It is the base class for PV Rows that will contain all the boiler plate code shared by sub classes like PVRowLine, or for instance PVRowRoof.
