pvfactors.timeseries.get_average_pvrow_outputs¶

pvfactors.timeseries.get_average_pvrow_outputs(df_registries, values=['q0', 'qinc', 'circumsolar_term', 'horizon_term', 'direct_term', 'irradiance_term', 'isotropic_term', 'reflection_term', 'horizon_band_shading_pct'], include_shading=True)[source]¶

For each pvrow surface (front and back), calculate the weighted average irradiance and shading values (weighted by length of surface).

Parameters:
  • df_registries (pandas.DataFrame) – timestamped and concatenated Array surface registries
  • values (list, optional) – list of column names from the surface registries to keep (default=``COLS_TO_SAVE``)
  • include_shading (bool, optional) – flag to decide whether to keep column indicating if surface has direct shading or not
Returns:

dataframe with averaged values and flags
Return type:

pandas.DataFrame
