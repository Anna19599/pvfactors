pvfactors.pvgeometry.PVGeometry.split_pvrow_geometry¶

PVGeometry.split_pvrow_geometry(idx_pvrow, line_shadow, pvrow_top_point, surface_side)[source]¶

Break up pv row line into two pv row lines, a shaded one and an unshaded one. This function requires knowing the pv row line index in the registry, the “shadow line” that intersects with the pv row, and the top point of the pv row in order to decide which pv row line will be shaded or not after break up.

Parameters:
  • idx_pvrow (int) – index of shaded pv row entry
  • line_shadow (shapely.LineString) – Line representing the “shadow line” intersecting with the pv row line
  • pvrow_top_point (shapely.Point) – the highest point of the pv row line (in the elevation direction)
  • surface_side (str) – surface side of the pvrow
Returns:
Return type:

None
