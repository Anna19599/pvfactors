pvfactors.pvgeometry.PVGeometry.cut_pvrow_geometry¶

PVGeometry.cut_pvrow_geometry(list_points, pvrow_index, side, count_segments=False)[source]¶

Break up pv row lines into multiple segments based on the list of points specified. This is the “discretization” of the pvrow segments. For now, it only works for pv rows.

Parameters:
  • list_points (list) – list of shapely.Point, breaking points for the pv row lines.
  • pvrow_index (index) – pv row index to specify the PV row to discretize; note that this could return multiple entries from the registry.
  • side (str) – only do it for one side of the selected PV row. This can only be ‘front’ or ‘back’.
  • count_segments (bool, optional) – Add count of segments (Default = False)
Returns:
Return type:

None
