mapper.VFMapperOrderedPVArray¶

class mapper.VFMapperOrderedPVArray[source]¶

This class maps the “view types” of the OrderedPVArray view_matrix property to some view factor calculation functions. For instance, the method to calculate the view factor between 2 PV row surfaces will be different than between a PV row surface and a ground one.
