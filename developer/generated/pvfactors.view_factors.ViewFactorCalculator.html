pvfactors.view_factors.ViewFactorCalculator¶

class pvfactors.view_factors.ViewFactorCalculator[source]¶

This class calculates the view factors based on the surface registry of Array class. Its only attribute is a mapping between “view types” and the “low-level” view factor calculation functions
