pvfactors.pvarray.ArrayBase¶

class pvfactors.pvarray.ArrayBase(n_pvrows, pvrow_height)[source]¶
ArrayBase exists for future developments of the model. It is the
base class for PV arrays that will contain all the boiler plate code shared by sub classes like Array, or for instance more complex PV array classes with varying GCR values, or non-flat ground, etc.
Parameters:
  • n_pvrows (int) – number of PV rows in PV array
  • pvrow_height (float) – height of PV rows
