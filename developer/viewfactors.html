viewfactors¶

The viewfactors sub-package of pvfactors implements the methods used to calculate view factors from pvfactors PV array objects.

calculator¶

Module with classes and functions to calculate views and view factors

VFCalculator

This calculator class will be used for the calculation of view factors for PV arrays

methods¶

Module with view factor calculation tools

VFTsMethods

This class contains all the methods used to calculate timeseries view factors from PV row elements to all the other elements of a PV array.
